This page is up to test a new version of the 1140px CSS Grid. It's using .css{user:agent;} to detect IE. So all grid styles can be in one style sheet. It's also using .css{user:agent;} to sniff whether the user agent is a mobile device or not. I'm using this to provide a mobile layout for devices that don't support media queries like WP7.
